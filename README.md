# Coffee Shop Sales Dashboard (Excel)
# An interactive Excel dashboard analyzing coffee shop sales across multiple store locations. 
# Built using pivot tables and pivot charts with slicers for Day Name and Month filtering. 
# Key insights include total sales ($698,812.33), footfall (149,116), average bill per person, hourly quantity trends, top 5 products by sales, weekday order patterns, size distribution, and category-wise sales breakdown across Astoria, Hell's Kitchen, and Lower Manhattan locations.
